### format - Keyword 종류

```bash
# solve -> 풀이에 성공한 문제
format: (solve) 백준 BFS/DFS 난이도 제목

# fail -> 풀이에 실패한 문제
format: (fail) 백준 BFS/DFS 난이도 제목

# improve[solve, fail] -> 복습한 문제
format: (improve[solve]) 백준 BFS/DFS 난이도 제목
```
