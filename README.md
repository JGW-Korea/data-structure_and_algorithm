### format - Keyword 종류

- `solve` : 풀이에 성공한 문제
- `fail` : 풀이에 실패한 문제
- `improve(solve, fail)` : 복습한 문제

### Solved.ac Tips

```(bash)
// 아직 풀어보지 않은 문제 중 난이도가 실버 5 ~ 골드 1까지의 모든 유형임과 동시에 1000명 이상이 정답을 맞춘 문제
*s5..g1 !@$me s#1000..

// 아직 풀어보지 않은 문제 중 난이도가 브론즈 3 ~ 실버 1까지의 시뮬레이션 문제임과 동시에 1000명 이상이 정답을 맞춘 문제
*b3..s1 !@$me s#1000.. #simulation
```
